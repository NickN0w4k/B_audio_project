# 🧠 KI-Audio-Restauration — Obsidian Vault

> **Ursprung:** NotebookLM-Projekt *„Refining the Synthetic Sound: AI Audio Restoration Architectures"*
> Exportiert am 29.04.2026

Öffne [[00_HOME]] als Startseite.

## Vault-Struktur

```
notebooklm-export/
├── 00_HOME.md          ← Startseite / Map of Content
├── README.md           ← Diese Datei
├── .obsidian/          ← Obsidian-Konfiguration
├── assets/             ← Bilder, Screenshots
├── doku/               ← 7 thematische Synthese-Artikel (Deutsch)
├── quellen/            ← 26+ extrahierte Quellen
│   ├── stem-separation/
│   ├── vocoder/
│   ├── denoising/
│   ├── mastering/
│   ├── paper/
│   ├── notebooklm-intern/
│   └── (übergreifende Quellen)
└── studio/             ← 16 Studio-Outputs
    ├── text-notes/     ← 5 extrahierte Text-Notes
    ├── infografiken/   ← Referenz auf nicht-extrahierbare Bilder
    └── videos/         ← Referenz auf Audio-Eklärvideos
```

## Features

- **Wikilinks** `[[Dateiname|Anzeigetext]]` für Vernetzung
- **YAML-Frontmatter** mit Tags in jeder Datei
- **Tag-Gruppen** im Graph-View: `#artefakt` `#tool` `#pipeline` `#mastering` `#quelle`
- **Map of Content** ([[00_HOME]]) als Einstiegspunkt
